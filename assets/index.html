What do these parameters mean?
corridor (m)
Total width of the corridor around the great-circle line A→B. The router pays a quadratic penalty for straying off the line, scaled to the corridor. Smaller = the route hugs the line tightly (and may fail in mountains). Larger = the router is allowed to detour further to find a viable path. Try 5 km for a strict line, 8–15 km for a doable one.
modes
Which OSM ways the router is allowed to use. foot includes paths and footways; bike wants surfaces a road bike tolerates; mtb opens up rough singletrack. Combinations widen the graph.
candidates
How many top suggestions to keep after scoring + deduplication. Country mode generates ~6× this number internally, then prunes near-duplicates so the cards you see are visually distinct lines.
elevation pts
Samples taken along each route for the elevation chart. 0 skips the lookup entirely (faster). Each batch of up to 100 points is one HTTP call to OpenTopoData (rate-limited to ~1/sec, so high values × many candidates = slower).
distance min/max km
Length range for country-mode candidate lines. The straight-line distance between start and end must fall in this band. Doesn't apply to "between A and B" mode.
border-to-border only
When ticked, country mode only proposes lines whose start and end both land in a strip near the country's bbox edges, on different sides — so every suggestion is a true country traversal. Set distance max generously (Switzerland's diagonal is ~360 km).
border strip km
How wide the "border strip" is. 25 km gives the sampler room to find graph nodes near the edge; smaller values force more extreme corner-to-corner candidates.
(internal) alpha
For "between A and B", the server runs A* with three different α values (0.5, 4, 12) to give qualitatively different alternatives — loose, balanced, strict.
(internal) score
Lower is easier. score = 100·(real_km/ref_km−1) + 5·long-edge km + 8·max-deviation km. Detour ratio dominates; long edges penalise unavoidable lake/glacier hops; max deviation flags lines that bulge wide.
